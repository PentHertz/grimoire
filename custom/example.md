# Example custom note

Drop any Markdown file into this `custom/` folder and re-run
`grimoire.py index` - it is indexed under the **custom** source and searchable
in the web UI alongside HackTricks, OWASP, PayloadsAllTheThings, etc.

Example searchable content: ssrf gopher redis filter bypass, kerberoast,
jwt none-alg, sudo GTFOBins, xss polyglot.
